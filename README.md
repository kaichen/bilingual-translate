# bilingual translate

网页翻译浏览器插件，双语对照呈现。

## 安装

```bash
pnpm install
pnpm build
```

构建结果位于 `.output/chrome-mv3`，可在浏览器扩展管理页面中以开发者模式加载。

## 参考

基于 FluentRead 改造重写，参考 Kiss Translator 功能实现

- https://github.com/Bistutu/FluentRead
- https://github.com/fishjar/kiss-translator
